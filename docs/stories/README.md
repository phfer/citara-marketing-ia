## Como Trabalhar

1. Selecione o epic
2. Vá para a pasta do sprint atual
3. Abra a story que vai trabalhar
4. Siga os acceptance criteria
5. Marque checkboxes conforme progresso
6. Atualize status quando completo

## Status Codes

- **Draft:** Story criada, não iniciada
- **Ready:** Validada pelo @po, pronta para dev
- **InProgress:** Em desenvolvimento
- **InReview:** Em revisão pelo @qa
- **Done:** Completa e aprovada

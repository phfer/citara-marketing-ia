# CONSTRUÇÃO DO LOGOTIPO PARA DESIGNERS

## Especificações Técnicas Gerais

**Grid:**
- Usar grid de 8px para alinhamento
- Proporção áurea (1:1.618) para composição

**Espaçamento:**
- Counter space (espaço negativo dentro das letras) ≥ 2px
- Espaço entre elementos = altura do 'x' da fonte

**Versões Necessárias:**
- `.SVG` - Vetorial master
- `.PNG` 2000px - Alta resolução
- `.PNG` 500px - Uso geral
- `.PNG` 250px - Thumbnails
- `.PNG` 32px - Favicon
- `.JPG` 300dpi - Impressão
- `.PDF` - Print profissional

**Variantes de Cor:**
1. Full color
2. Positivo (Dark Navy sobre branco)
3. Negativo (Branco sobre Dark Navy)
4. Monocromático (Preto sobre branco)
5. Branco sobre fundo colorido
